## 功能說明

透過 JWT Token 驗證使用者 Session 是否有效。

## 測試步驟

1. API：GET `/api/auth/session` 帶 JWT Token
2. 確認回傳使用者資訊

## 預期結果

- 帶有效 Token 回傳 200 + 使用者資訊
- 無 Token 或過期 Token 回傳 401

API 回傳 200，Session 驗證正常。
