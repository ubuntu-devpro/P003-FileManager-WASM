## 失敗案例

### ❌ FM-002：新增資料夾

**問題：** API 通過，但 UI 對話框內 input 欄位定位失敗
**原因：** `wait_for_selector` 找到 dialog 容器而非內部的 input，`.fill()` 無法作用於非輸入元素
**對應方法：** 改用 `page.locator(".rz-dialog input[type='text']")` 精確定位

### ❌ FM-003：重新命名

**問題：** API 回傳 400，請求 body 格式不符
**原因：** rename 端點預期的欄位名稱可能與測試腳本不同
**對應方法：** 確認 Controller 的 DTO 欄位名稱後修正

### ❌ FM-004：移動檔案

**問題：** API 回傳 400；UI「移動」按鈕 disabled
**原因：** UI 需先勾選檔案 checkbox 啟用按鈕；API 請求格式不符
**對應方法：** UI 加入勾選 checkbox 步驟；API 確認欄位名稱

### ❌ FM-005：刪除

**問題：** 同 FM-004
**原因：** 同上
**對應方法：** 同上

### ❌ FM-011：搜尋

**問題：** API 回傳 400，UI 功能正常
**原因：** search 端點預期的欄位名稱可能與測試腳本不同
**對應方法：** 確認 Controller 的 DTO 欄位名稱後修正
