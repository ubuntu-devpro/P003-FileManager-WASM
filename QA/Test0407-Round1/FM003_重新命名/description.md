## 功能說明

選取檔案或資料夾後，可透過按鈕或右鍵選單觸發重新命名對話框。

## 測試步驟

1. API：PATCH `/api/files/rename` 帶 `{ path, newName }` + JWT Token
2. UI：開啟檔案列表
3. UI：嘗試點擊重新命名按鈕或右鍵觸發選單
4. UI：等待對話框出現並截圖

## 預期結果

- API 回傳 200，檔案重新命名成功
- UI 對話框正常彈出

## 實際結果

API 回傳 400（請求格式錯誤，可能欄位名稱不符）。UI 成功截到對話框畫面。
