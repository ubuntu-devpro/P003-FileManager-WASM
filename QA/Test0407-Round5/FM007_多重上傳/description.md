## 功能說明

使用者可一次選擇多個檔案進行上傳，上傳對話框顯示所有已選檔案清單。

## 測試步驟

1. API 測試：POST /api/upload（與 FM-006 相同端點），確認支援多檔
2. UI 測試：點擊工具列「上傳」按鈕
3. 確認 file input 具有 `multiple` 屬性
4. 選擇 3 個檔案，確認檔案清單正確顯示所有檔案名稱與大小
5. 點擊「上傳」按鈕
6. 確認全部上傳完成，顯示成功通知，所有檔案出現在列表中

## 預期結果

- API 回傳 200，多檔上傳成功
- UI 對話框正確列出所有已選檔案
- 上傳完成後顯示「已上傳 3/3 個檔案」通知

## 實際結果

API 回傳 200，多檔上傳成功。UI 上傳對話框中選擇 3 個檔案後，清單正確顯示所有檔案名稱與大小。點擊上傳後全部成功，檔案列表自動刷新，3 個新檔案皆出現在列表中。
