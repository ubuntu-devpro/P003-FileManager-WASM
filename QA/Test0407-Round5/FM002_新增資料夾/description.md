## 功能說明

使用者可透過「新增資料夾」按鈕建立新資料夾，系統彈出對話框供使用者輸入資料夾名稱。

## 測試步驟

1. API 測試：POST /api/folders，確認可建立資料夾
2. UI 測試：點擊「新增資料夾」按鈕
3. 確認對話框彈出，包含文字輸入欄位
4. 輸入資料夾名稱「QA_TestNewFolder」
5. 點擊「建立」按鈕
6. 確認檔案列表中出現新建的資料夾

## 預期結果

- API 回傳 200，資料夾建立成功
- UI 點擊按鈕後彈出對話框，輸入名稱後點擊建立
- 建立完成後檔案列表自動刷新，顯示新資料夾

## 實際結果

API 回傳 200，資料夾建立成功。UI 點擊「新增資料夾」按鈕後，對話框正常彈出，輸入「QA_TestNewFolder」後點擊建立，檔案列表自動刷新並顯示新建的資料夾。
