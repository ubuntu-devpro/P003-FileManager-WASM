## 功能說明

使用者可勾選多個檔案後，點擊工具列「下載」按鈕一次觸發多檔下載。

## 測試步驟

1. UI 測試：進入 devpro.com.tw 資料夾
2. 勾選 3 個測試檔案（dl_multi_1.txt、dl_multi_2.txt、dl_multi_3.txt）
3. 確認狀態列顯示「已選取 3 項」，工具列下載按鈕啟用
4. 點擊工具列「下載」按鈕
5. 確認顯示「下載 — 已開啟 3 個下載」通知

## 預期結果

- 可勾選多個檔案，狀態列正確顯示已選取數量
- 點擊下載按鈕後觸發多檔下載，顯示下載數量通知

## 實際結果

成功勾選 3 個檔案，3 個檔案列高亮顯示。點擊工具列下載按鈕後，系統逐一觸發下載，右上角顯示「下載 — 已開啟 3 個下載」通知。
