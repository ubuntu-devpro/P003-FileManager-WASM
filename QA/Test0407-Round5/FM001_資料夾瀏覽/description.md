使用者登入後可瀏覽檔案列表，以表格形式顯示資料夾與檔案，並可雙擊進入子資料夾。

## 測試步驟

1. API 測試：GET /api/files，確認回傳檔案清單
2. UI 測試：登入後開啟首頁，確認檔案列表表格存在
3. UI 測試：雙擊子資料夾，確認導航至子目錄並顯示內容

## 預期結果

- API 回傳 200，包含檔案清單資料
- UI 顯示檔案列表表格，含名稱、大小、修改時間、操作欄位
- 雙擊資料夾後，breadcrumb 更新為子目錄路徑，列表顯示子目錄內容
