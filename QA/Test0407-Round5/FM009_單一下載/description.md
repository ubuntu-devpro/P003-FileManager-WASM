## 功能說明

使用者可透過行內下載按鈕 ⬇️ 或工具列下載按鈕下載單一檔案。

## 測試步驟

1. API 測試：GET /api/files/download，確認回傳 200 及檔案內容
2. UI 測試：進入 devpro.com.tw 資料夾
3. 勾選目標檔案
4. 點擊行內下載按鈕 ⬇️，確認下載觸發
5. 點擊工具列「下載」按鈕，確認顯示「已開啟 N 個下載」通知

## 預期結果

- API 回傳 200，下載功能正常
- UI 點擊下載按鈕後觸發檔案下載
- 工具列下載顯示下載通知
