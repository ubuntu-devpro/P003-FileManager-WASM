## 功能說明

使用者可勾選檔案後進行刪除操作，系統會彈出確認對話框防止誤刪。

## 測試步驟

1. API 測試：DELETE /api/files，傳送 {paths[]}
2. UI 測試：進入 devpro.com.tw 資料夾
3. 勾選目標檔案 checkbox
4. 點擊刪除按鈕
5. 確認對話框彈出，顯示「確定要刪除「檔名」嗎？」及「此操作無法復原」提示
6. 點擊「刪除」按鈕確認刪除
7. 觀察刪除結果與通知

## 預期結果

- API 回傳 200，刪除成功
- UI 點擊刪除按鈕後彈出確認對話框
- 確認後檔案被刪除，顯示「成功 - 已刪除」通知
- 檔案列表自動刷新

## 實際結果

API 回傳 200，刪除成功。UI 點擊刪除按鈕後正確彈出確認對話框，顯示檔案名稱與警告訊息。點擊確認後檔案成功刪除，右上角顯示「成功 - 已刪除」通知，檔案列表自動刷新。
