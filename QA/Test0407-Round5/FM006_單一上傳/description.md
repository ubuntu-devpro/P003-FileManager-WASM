## 功能說明

使用者可透過上傳按鈕選擇單一檔案進行上傳，上傳完成後檔案列表自動刷新。

## 測試步驟

1. API 測試：POST /api/upload，確認上傳功能正常
2. UI 測試：點擊工具列「上傳」按鈕
3. 確認上傳對話框彈出，包含檔案選擇區域
4. 選擇一個檔案，確認檔案名稱與大小顯示在列表中
5. 點擊「上傳」按鈕
6. 確認上傳完成，顯示成功通知，檔案出現在列表中

## 預期結果
