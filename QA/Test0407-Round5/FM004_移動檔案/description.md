## 功能說明

使用者可選取檔案後點擊「移動」按鈕，系統彈出移動對話框，列出可選的目標資料夾。選擇目標後確認，檔案即移動至指定資料夾。

## 測試步驟

1. API 測試：PATCH /api/files/move，傳送 {sourcePaths[], destinationPath}，確認回傳 200
2. UI 測試：進入 devpro.com.tw 資料夾
3. 勾選目標檔案 checkbox
4. 確認工具列「移動」按鈕啟用
5. 點擊「移動」按鈕，確認移動對話框彈出
6. 對話框中顯示可選的子資料夾列表
7. 點擊選擇目標資料夾（高亮顯示），底部顯示「移動到：資料夾名」
8. 點擊「移動」按鈕確認
9. 確認成功通知「已移動 1 個項目」
10. 確認檔案從原目錄消失
11. 進入目標資料夾，確認檔案已移入

## 預期結果

- API 回傳 200，移動成功
- UI 移動對話框正確列出可選資料夾
- 移動完成後顯示成功通知，檔案從原目錄消失並出現在目標資料夾中

## 實際結果

API 回傳 200，移動成功。UI 點擊移動按鈕後對話框正常彈出，列出當前目錄下的子資料夾。選擇 MoveTarget 資料夾後高亮顯示，底部提示「移動到：MoveTarget」。點擊確認後顯示「成功 — 已移動 1 個項目」通知，檔案從原目錄消失，進入 MoveTarget 確認檔案已成功移入。
