## 功能說明

使用者可對檔案或資料夾進行重新命名，點擊操作欄的 ✏️ 按鈕後系統彈出對話框供輸入新名稱。

## 測試步驟

1. API 測試：PATCH /api/files/rename，傳送 {currentPath, newName}，確認回傳 200
2. UI 測試：在檔案列表中找到目標檔案
3. 點擊該檔案的重新命名按鈕 ✏️
4. 確認對話框彈出，顯示目前檔名
5. 輸入新名稱「renamed_file.txt」
6. 點擊「確認」按鈕
7. 確認檔案列表更新，顯示新檔名

## 預期結果

- API 回傳 200，重新命名成功
- UI 彈出對話框，預填目前檔名
- 輸入新名稱確認後，檔案列表顯示更新後的檔名

## 實際結果

API 回傳 200，重新命名成功。UI 點擊 ✏️ 按鈕後對話框正常彈出，預填目前檔名。輸入「renamed_file.txt」後點擊確認，檔案列表自動刷新，顯示重新命名後的檔名。
