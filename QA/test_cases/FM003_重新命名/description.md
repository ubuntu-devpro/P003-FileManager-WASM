## 功能說明
透過右鍵選單或工具列，選擇「重新命名」，彈出對話框，輸入新名稱後完成重新命名。

## 測試步驟
1. 點擊任一檔案的「重新命名」按鈕（✏️）
2. 對話框彈出，輸入新名稱
3. 點擊確認
4. 觀察檔案名稱是否更新

## 預期結果
- 重新命名對話框正常彈出
- 對話框包含「新名稱」標籤和輸入框
- 確認後檔案名稱更新

## 實際結果
✅ PASS — 對話框顯示「重新命名」標題和「新名稱」輸入框，符合預期。
