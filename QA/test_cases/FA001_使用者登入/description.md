## 功能說明
輸入帳號密碼，系統驗證後核發 JWT Token，自動登入至首頁。

## 測試步驟
1. 訪問 http://localhost:5000，自動導向 /login
2. 輸入 Email 和密碼
3. 點擊登入
4. 成功後自動跳轉至首頁
