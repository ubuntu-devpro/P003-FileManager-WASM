## 功能說明
點擊「登出」按鈕，系統清除 JWT Token，跳轉回登入頁。

## 測試步驟
1. 以 admin 身份登入
2. 點擊右上角「登出」按鈕
3. 觀察是否跳轉至登入頁

## 預期結果
- 點擊登出後跳轉至 /login
- JWT Token 被清除
