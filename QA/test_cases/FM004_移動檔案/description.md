## 功能說明

選擇檔案或資料夾後，點擊「移動」，彈出對話框選擇目標資料夾，完成移動。

## 測試步驟

1. 勾選要移動的檔案
2. 點擊「移動」按鈕
3. 對話框彈出，選擇目標資料夾
4. 點擊確認
