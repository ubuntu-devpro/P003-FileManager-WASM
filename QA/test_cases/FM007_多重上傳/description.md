## 功能說明
上傳對話框支援多檔選擇（HTML input multiple 屬性），可一次上傳多個檔案。

## 測試步驟
1. 點擊「上傳」按鈕
2. 上傳對話框彈出
3. 檢查 input[type=file] 的 multiple 屬性

## 預期結果
- 上傳 input 支援 multiple 屬性
- 可選擇多個檔案

## 實際結果
✅ PASS — `input[type=file].multiple = true`，支援多檔選擇
