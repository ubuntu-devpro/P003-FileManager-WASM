## 功能說明

勾選多個檔案後，點擊「下載」按鈕，系統將多個檔案打包成 ZIP 供下載。

## 測試步驟

1. 在檔案列表勾選多個檔案
2. 點擊「下載」按鈕
3. 觀察 ZIP 下載行為
