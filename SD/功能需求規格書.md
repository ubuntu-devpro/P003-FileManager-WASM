| 資料儲存 | 檔案系統（無資料庫）| — |

### 2.1 無 SignalR 決策

**原因：** Blazor Server 的 SignalR 電路在 CI/headless 自動化環境不穩定，導致 Playwright 截圖失敗。

**驗證結果：** P002（Blazor Server）踩坑後，確定更換為 Blazor WASM，自動化測試穩定通過。

## 4. 認證與權限詳細規格

### 4.1 預設帳號

| Email | 密碼 | 角色 | 可見範圍 |
|-------|------|------|---------|
| admin@devpro.com.tw | admin123 | Admin | `/home/devpro/data/*`（全部網域）|
| johnny@sinopac.com | johnny123 | User | `/home/devpro/data/sinopac.com/*` |
| user@others.com | user123 | User | `/home/devpro/data/others.com/*` |

### 4.2 JWT 規格

| 項目 | 規格 |
|------|------|
| Algorithm | HS256 |
| Issuer | FileManager.Server |
| Audience | FileManager.Client |
| 過期時間 | 24 小時 |
| Claims | email, role (Admin/User), domain |

### 4.3 網域過濾邏輯

- Admin：可見 `/home/devpro/data` 下的所有子目錄和檔案
- 一般使用者：只能見 `/home/devpro/data/{domain}/` 下的內容
- 超出自己網域的檔案完全無法存取（API 層阻擋）
